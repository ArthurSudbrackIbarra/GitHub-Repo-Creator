# GitHub-Repo-Creator

Automatically create GitHub repositories.

## Commands

```sh
grc create <path_to_yaml_file> # Creates a repository.
```

```sh
grc template <template_type> # Generates a template YAML file.
```

```sh
grc set-token <access_token> # Sets the user access token.
```
